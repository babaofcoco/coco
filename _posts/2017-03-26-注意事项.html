今天是2017年3月26日，在家闲得无聊，琢磨着怎样创作自己的blog。可是，时间太久竟然不知道怎么使用git了。下面我把如何使用github做个简单的总结，主要目的是提供github如何使用。

  1. 刚打开家里的电脑，居然没有安装git。。。。惭愧家里的电脑多是用来看电视剧，看电影了

    2. 下载git，并安装，我下载的地址是：https://git-scm.com/download/win

  2. 克隆一下已有代码。git clone -b gh-pages https://github.com/babaofcoco/coco.git
  3. 查看修改了哪些文件。git status
  4. 添加远程不存在的git文件。git add .
  5. 提交修改。git commit -m "memo"
  6. 更新到远程服务器上。git push
  7. 移除，删除文件。git rm 文件
  8. 查看所有分支。git branch -r
  9. 创建或者获取某一个分支，git checkout 分支; git check --orphan gh-pages
  10. 在使用的过程中不想每次都输入用户名和密码，需要用到ssh-keygen

    11. 1, ssh-keygen -t rsa -C "2435644284@qq.com"

    2，将生成的id-rsa.pub里面的内容复制，并拷贝到https://github.com/settings/keys页面下，New SSH key

    3, 检查有没有成功。git -T git@github.com

    4, 除此之外添加如下两句：git config --global user.name "babaofcoco";git config --global user.email "2435644284@qq.com"


我使用github的一般步骤
  1. 首先应该利用pull来更新本地的代码: pull
  2. 当修改了代码使用git add .; git commit -m "commet"; git push;来上传代码
